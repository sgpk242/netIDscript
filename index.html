Netidscript

A script to retrieve netID's from Notre Dame's Directory online application from last names.

Welcome to My Bookmarklet Page.

I was experimenting with bookmarklets for a while when I had a university project to locate the user IDs of faculty so that they could be entered for a department election. The only way to find the user ID was by entering the faculty member's name into my university's online directory form - only one name could be submitted at a time. To expedite the process of locating over 200 faculty member's IDs, I decided to look into bookmarklets.

This text would be entered into the URL of the bookmarklet:

javascript:(function(){   
   document.body.appendChild(document.createElement('script')).src='http://sgpk242.github.io/netIDscript/final.js';
})();

I used an 'alert' function to test this (code [here]'http://sgpk242.github.io/netIDscript/test.js'). Final.js can be found [here]'http://sgpk242.github.io/netIDscript/final.js'.

Although I found them to be a really interesting tool, I decided to stop pursuing this project when I learned that the Content Security Policy (2015) prevented me from using jQuery and my files hosted on this site in my bookmarklets on other sites.
